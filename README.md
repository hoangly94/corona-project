## Scripts

| Script | Desc |
| ---- | ---- |
|`$ yarn start`| Start a simple webpack server |
|`$ yarn dev`| Create a server to development at port 5000 |
|`$ yarn storybook`| Start Storybook with the stories imported |
|`$ yarn flow`| Validate the flow types |
